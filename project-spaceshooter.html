📖 项目简介

这是一个使用 Python 和 Pygame 开发的经典太空射击游戏项目。通过学习和参考 GitHub 上的高星开源项目 spaceShooter (524⭐)，我深入理解了游戏开发的核心概念，包括游戏循环、碰撞检测、状态管理、音效处理等。

项目展示了我的自主学习能力、代码理解能力和游戏开发基础， 体现了在 AI 编程工具辅助下快速掌握新技术并实现完整项目的能力。

🎯 核心功能

❤️

生命值系统

玩家拥有 3 条生命，被敌机撞击或被子弹击中时扣除生命值，生命值耗尽则游戏结束。

📊

分数排行榜

实时显示当前得分，击落敌机获得分数，支持历史最高分记录。

🎁

道具系统

随机掉落道具：护盾（临时无敌）、连射（同时发射多颗子弹）、生命恢复等。

💥

碰撞检测

精确的矩形碰撞检测系统，处理玩家与敌机、子弹与敌机、道具与玩家的碰撞。

🔊

音效系统

包含射击音效、爆炸音效、背景音乐等完整的音频体验。

✨

动画效果

玩家飞机、敌机、子弹、爆炸等精灵动画，流畅的视觉效果。

🛠️ 技术栈

Python 3.x Pygame SDL 面向对象编程 碰撞检测算法 状态机 Trae Cursor GitHub

🚀 运行方式

# 1. 安装依赖 pip install pygame # 2. 运行游戏 python space_shooter.py # 3. 控制方式 ← → : 左右移动 空格键 : 发射子弹 ESC : 退出游戏
